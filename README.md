# nonebot-plugin-laofei-tools

老肥工具箱 - NoneBot2 插件

## 功能

### 积分系统
- **签到**：每日签到获取积分和经验，连续签到有加成
- **银行**：存取积分，每天0点自动计算利息（1%）
- **娱乐**：抽奖、猜数字、打劫、抢银行
- **排行榜**：查看积分前十名

### 宠物系统
- **领养宠物**：5种宠物可选，各有独特天赋
- **宠物养成**：散步获取经验和道具、抚摸提升好感度、喂食恢复体力
- **宠物PK**：与他人宠物对战，属性影响胜率
- **商店系统**：购买食物和配饰，配饰提升宠物属性
- **背包系统**：管理食物和配饰道具

### 搜图功能
- **以图搜图**：基于 soutubot.moe 的以图搜图功能（需超级用户开启）

## 安装

```bash
pip install nonebot-plugin-laofei-tools
```

## 使用

### 积分指令

| 指令 | 说明 |
|------|------|
| `签到` / `打卡` | 每日签到获取积分和经验 |
| `积分` / `查积分` | 查看自己的积分信息 |
| `积分排行` / `排行榜` | 查看积分前十名 |
| `转账 积分 @某人` | 将积分转账给他人 |
| `打劫 @某人` | 打劫他人积分（需50积分以上） |

### 银行指令

| 指令 | 说明 |
|------|------|
| `存入银行 积分` | 将积分存入银行 |
| `取出银行 积分` | 将积分从银行取出 |
| `抢银行` | 抢银行获取积分（需50积分以上） |

### 娱乐指令

| 指令 | 说明 |
|------|------|
| `抽奖 积分` | 消耗积分抽奖（范围10-500） |
| `猜数字 积分` | 开始猜数字游戏（范围10-500） |
| `我猜 数字` | 猜数字（范围1-100，6次机会） |

### 宠物指令

| 指令 | 说明 |
|------|------|
| `我的宠物` | 查看宠物信息 / 领养宠物 |
| `领养 宠物名` | 领养指定宠物 |
| `宠物散步` | 消耗体力散步获取经验和道具 |
| `宠物抚摸` | 每日抚摸提升好感度 |
| `宠物喂食 食物名` | 喂食恢复体力和好感度 |
| `宠物pk @某人` | 与他人宠物PK对战 |
| `宠物商店` | 查看商店商品（图片展示） |
| `购买 商品名` | 使用积分购买商品 |
| `宠物佩戴 配饰名` | 佩戴配饰提升属性 |
| `宠物背包` | 查看道具背包 |
| `宠物帮助` | 查看宠物系统帮助 |

### 宠物种类

| 宠物 | 幸运 | 武力 | 天赋 |
|------|------|------|------|
| Doro | 10 | 10 | 欧润橘：好感提升速度1.2倍 |
| 香企鹅 | 20 | 10 | 咕咕嘎嘎：食物恢复比例1.4倍 |
| 刀盾狗 | 0 | 20 | 我的刀盾：PK胜率加5% |
| 耄耋 | 10 | 15 | 哈气：散步掉落道具概率加10% |
| 奶龙 | 15 | 15 | 我是奶龙：初始体力多20 |

### 管理指令

| 指令 | 说明 |
|------|------|
| `开启积分` | 超级用户开启群聊积分系统 |
| `关闭积分` | 超级用户关闭群聊积分系统 |
| `功能` | 查看所有功能列表 |
| `功能 功能名` | 查看指定功能的详细说明 |

## 配置

在 `.env` 文件中添加：

```env
# 可选：配置默认开启搜图的群聊列表
LAOFEI_SEARCH_ENABLED_GROUPS=["123456789", "987654321"]
```

## 数据存储

插件数据存储在 `data/laofei_tools/` 目录下：
- `user_points.json` - 用户积分数据
- `pet_data.json` - 宠物数据
- `pet_inventory.json` - 宠物背包数据
- `enabled_groups.json` - 已开启搜图的群聊
- `points_disabled_groups.json` - 已关闭积分系统的群聊

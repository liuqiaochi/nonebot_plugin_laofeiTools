# nonebot-plugin-longge-tools

龙哥工具箱 - NoneBot2 插件，集成积分系统、宠物养成、以图搜图、图片超分等功能。

## 项目架构

```
nonebot_plugin_laofei_tools/
├── __init__.py              # 主入口：lg帮助指令、定时任务、启动初始化
├── config.py                # 共享配置：群聊开关、数据目录
├── image/                   # 静态图片资源（抽签签文图、宠物图、商品图）
├── common/                  # 通用模块
│   ├── points_commands.py   # 积分系统指令（签到、抽签、积分查询、新手大礼包）
│   ├── points_data.py       # 积分数据管理、银行利息、抽签逻辑
│   ├── lottery_pool.py      # 幸运奖池系统
│   ├── enhance.py           # 图片超分功能（lg超分）
│   ├── restart.py           # 重启bot功能
│   └── utils.py             # 公共工具函数（图片下载等）
├── pet/                     # 宠物模块
│   ├── pet_commands.py      # 宠物系统全部指令
│   ├── pet_data.py          # 宠物数据管理、体力/经验/属性
│   └── shop_image.py        # 商店图片生成
└── search/                  # 搜图模块
    ├── commands.py          # 搜图指令、管理指令、重启通知
    └── soutubot.py          # soutubot.moe API 客户端
```

## 安装

```bash
pip install nonebot-plugin-longge-tools
```

## 功能一览

### 积分系统

| 指令 | 别名 | 说明 |
|------|------|------|
| `签到` | 打卡、积分签到、日常签到 | 每日签到获取积分和经验，连续签到有加成 |
| `积分` | 查积分、我的积分 | 查看自己的积分信息 |
| `抽签` | 今日运气、今日抽签、今日气运 | 每日抽签（撞大运概率2%） |
| `新手大礼包` | 领取新手大礼包 | 领取新手大礼包 |

> 以下功能暂未启用：转账、银行存取、抽奖、猜数字、排行榜、PK对战

### 宠物系统

| 指令 | 别名 | 说明 |
|------|------|------|
| `我的宠物` | 宠物 | 查看宠物信息 / 领养宠物 |
| `领养 宠物名` | 宠物领养、领养宠物 | 领养指定宠物（5种可选） |
| `宠物散步` | 宠物溜达、宠物出门 | 消耗体力散步获取经验和道具 |
| `宠物抚摸` | 摸宠物、疼爱宠物 | 每日抚摸提升好感度 |
| `宠物喂食 食物名` | 宠物喂养、喂养宠物 | 喂食恢复体力和好感度 |
| `宠物pk @某人` | 宠物PK、宠物Pk、宠物pK | 与他人宠物PK对战 |
| `宠物商店` | 宠物店、宠物店铺 | 查看商店商品（图片展示） |
| `购买 商品名` | — | 使用积分购买商品 |
| `宠物佩戴 配饰名` | 宠物穿戴、宠物装备、使用配饰、穿配饰 | 佩戴配饰提升属性 |
| `宠物背包` | — | 查看道具背包 |
| `出售 商品名` | — | 出售背包中的道具 |
| `宠物打工` | — | 宠物打工赚取积分 |
| `飞龙探云手` | 宠物偷窃、我偷 | 偷窃其他用户积分 |
| `宠物弃养` | — | 弃养当前宠物（需二次确认） |
| `宠物改名` | — | 给宠物起名字 |
| `宠物帮助` | — | 查看宠物系统帮助 |

| 刀盾狗 | 0 | 20 | 我的刀盾：PK胜率加5% |
| 耄耋 | 10 | 15 | 哈气：散步掉落道具概率加10% |
| 奶龙 | 15 | 15 | 我是奶龙：初始体力多20 |

### 搜图功能

| 指令 | 说明 |
|------|------|
| `lg搜图` | 引用图片进行以图搜图（基于 soutubot.moe，需开启） |
| `搜图帮助` | 查看搜图帮助 |

> 搜图功能需超级用户在群聊中发送 `开启lg搜图` 开启。

### 图片工具

| 指令 | 说明 |
|------|------|
| `lg超分` | 引用图片进行高清化处理，让模糊图片变清晰 |

> 使用 OpenCV 图像增强流水线（双边滤波去噪 → 细节增强 → USM锐化 → CLAHE对比度增强）。

### 管理指令（超级用户）

| 指令 | 说明 |
|------|------|
| `开启积分` | 开启群聊积分系统 |
| `关闭积分` | 关闭群聊积分系统 |
| `开启lg搜图` | 开启群聊搜图功能 |
| `关闭lg搜图` | 关闭群聊搜图功能 |
| `重启bot` | 重启Bot（重启后自动发送成功通知） |
| `#准备重启` | 向所有群发送重启通知 |

### 帮助指令

| 指令 | 别名 | 说明 |
|------|------|------|
| `lg帮助` | 龙哥帮助、lg help | 查看所有功能总览 |
| `搜图帮助` | — | 查看搜图帮助 |
| `宠物帮助` | — | 查看宠物系统帮助 |

## 配置

在 `.env` 文件中添加：

```env
# 可选：配置默认开启搜图的群聊列表
LONGGE_SEARCH_ENABLED_GROUPS=["123456789", "987654321"]
```

## 数据存储

插件数据存储在 `data/laofei_tools/` 目录下：

| 文件 | 说明 |
|------|------|
| `user_points.json` | 用户积分、签到、抽签数据 |
| `guess_games.json` | 猜数字游戏数据 |
| `enabled_groups.json` | 已开启搜图的群聊 |
| `points_disabled_groups.json` | 已关闭积分系统的群聊 |
| `enhance/` | 图片超分临时文件（处理后自动清理） |

# nonebot-plugin-laofei-tools

老肥工具箱 - NoneBot2 插件

## 功能

### 搜图功能
- **以图搜图**：基于 soutubot.moe 的以图搜图功能

### 积分系统
- **签到**：每日签到获取积分和经验，连续签到有加成
- **银行**：存取积分，每天计算利息
- **娱乐**：抽奖、猜数字、打劫、抢银行

## 特点

- 搜图功能默认关闭，需超级用户手动开启
- 积分系统数据持久化存储

## 安装

```bash
pip install nonebot-plugin-laofei-tools
```

## 使用

### 超级用户指令

| 指令 | 说明 |
|------|------|
| `开启lf搜图` | 在当前群聊开启搜图功能 |
| `关闭lf搜图` | 在当前群聊关闭搜图功能 |

### 搜图指令

| 指令 | 说明 |
|------|------|
| `lf搜图` | 引用一张图片进行搜索（需先开启功能） |

### 积分指令

| 指令 | 说明 |
|------|------|
| `签到` / `打卡` | 每日签到获取积分和经验 |
| `积分` / `查积分` | 查看自己的积分信息 |
| `转账 积分 @某人` | 将积分转账给他人 |
| `打劫 @某人` | 打劫他人积分（需50积分以上） |

### 银行指令

| 指令 | 说明 |
|------|------|
| `存入银行 积分` | 将积分存入银行 |
| `取出银行 积分` | 将积分从银行取出 |
| `抢银行` | 抢银行获取积分（需50积分以上） |

### 娱乐指令

| 指令 | 说明 |
|------|------|
| `抽奖 积分` | 消耗积分抽奖，随机获取0-3倍积分（范围10-500） |
| `猜数字 积分` | 开始猜数字游戏（范围10-500） |
| `我猜 数字` | 猜数字（范围1-100，6次机会） |

### 帮助指令

| 指令 | 说明 |
|------|------|
| `功能` | 查看所有功能列表 |
| `功能 功能名` | 查看指定功能的详细说明 |

## 配置

在 `.env` 文件中添加：

```env
# 可选：配置默认开启搜图的群聊列表
LAOFEI_SEARCH_ENABLED_GROUPS=["123456789", "987654321"]
```

## 数据存储

插件数据存储在 `data/laofei_tools/` 目录下：
- `user_points.json` - 用户积分数据
- `enabled_groups.json` - 已开启搜图的群聊
- `guess_games.json` - 进行中的猜数字游戏

## 依赖

- nonebot2 >= 2.0.0
- nonebot-adapter-onebot >= 2.0.0
- httpx >= 0.24.0
- pillow >= 10.0.0

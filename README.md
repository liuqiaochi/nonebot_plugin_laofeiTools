# nonebot-plugin-laofei-tools

老肥工具箱 - NoneBot2 插件

## 功能

- **以图搜图**：基于 soutubot.moe 的以图搜图功能

## 特点

- 群聊专用，私聊不可用
- 默认关闭，需超级用户手动开启
- 结果以合并转发形式展示（图片+信息）

## 安装

```bash
pip install nonebot-plugin-laofei-tools
```

## 使用

### 超级用户指令

| 指令 | 说明 |
|------|------|
| `开启lf搜图` | 在当前群聊开启搜图功能 |
| `关闭lf搜图` | 在当前群聊关闭搜图功能 |

### 搜图指令

| 指令 | 说明 |
|------|------|
| `lf搜图` | 引用一张图片进行搜索（需先开启功能） |

### 使用示例

```
[超级用户] 开启lf搜图
Bot: ✅ 已开启本群搜图功能

[用户引用一张图片]
[用户] lf搜图
Bot: [合并转发消息，包含搜索结果]
```

## 配置

在 `.env` 文件中添加：

```env
# 可选：配置默认开启搜图的群聊列表
LAOFEI_SEARCH_ENABLED_GROUPS=["123456789", "987654321"]
```

## 依赖

- nonebot2 >= 2.0.0
- nonebot-adapter-onebot >= 2.0.0
- httpx >= 0.24.0
- pillow >= 10.0.0
